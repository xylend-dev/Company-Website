### Usage

**Requirements**

- Node(ˆv8.0)
- npm (ˆv5) or yarn(ˆv1.12)
- git

clone the repo
`git clone https://github.com/devconzm/devconzm.git`
`cd devconzm`

**Install dependencies**

`npm install`
or
`yarn`

**Run**

`npm start`
or
`yarn start`
